# Menu Layout — Combinators

A simple HTML and CSS project that demonstrates the use of CSS combinator selectors. The project contains a navigation menu and applies styling using descendant, child, and adjacent sibling combinators.

## Features

- Navigation menu built with HTML
- Descendant combinator selector
- Child combinator selector
- Adjacent sibling selector
- External CSS styling
- Simple and clean structure

## Technologies Used

- HTML5
- CSS3

## Project Structure

```text
menu-layout-combinators/
├── index.html
├── styles.css
└── README.md
```

## CSS Concepts Demonstrated

### Descendant Combinator
Selects all `<a>` elements inside the `.menu` navigation.

### Child Combinator (`>`)
Selects direct `<li>` children of the `<ul>` element and applies padding.

### Adjacent Sibling Combinator (`+`)
Selects the list item immediately following another list item and applies a background color.
